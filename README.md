# AnaxDb
An encrypted non-linear database based on Pandas

### Install AnaxDb

<code>
git clone https://github.com/abrahamrhoffman/AnaxDb.git<br>
cd AnaxDb<br>
pip install -r requirements.txt<br>
</code>

### Usage

<code>
import anaxdb
</code>
